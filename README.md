# Disco Romana

An engine for generated text-based games using conversational RPG mechanics similar to [Disco Elysium](https://discoelysium.fandom.com/wiki/Skills). Current example set in the Marius v. Sulla, in Late Roman Republic (~130-27 BC). All gameplay happens through conversations -- no direct combat, no resource spreadsheets. Political outcomes emerge from dialogue choices, factional alignments, and reputation.

BUG: Generated convos are not properly sequenced

Conversations and characters are procedurally generated by LLMs via [hankweave](./hankweave-runtime) orchestration. 

## Game Mechanics

The basic game design goals here are:
- Create a *fully deterministic* (subject to initial rng seed) dialogue based game, where all of the dialogue are generated by LLM. No plans for incorporating non-deterministic LLM conversational dialogue yet 
- Create text-based adventure game where dialogue choices *feel consequential*, and immerses the player in the game world.
- A purely text-based *progression system* that fulfills the actual videogame end of this bargain
- Teaches players about historical periods, the characters involved, the politics between them and their factions

Weak outcome: similar to "Choose your adventure" books, which does not give reader the consequence of choice so much as letting them read a non-linear storyline
Strong outcome: players actually feel like a "player" of world historical events (some imagination required)
